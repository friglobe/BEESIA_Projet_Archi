🚀 ÉTAPES RAPIDES
──────────────────

1. Lire le guide SEANCE-4-GUIDE-COMPLET.md

2. Charger la base de données:
   docker-compose exec db mysql -u root -proot cv_platform < bdd-cv.sql

3. Créer les APIs dans app/api/:
   - config.php
   - login.php
   - register.php
   - cv-form.php
   - profiles.php
   - convocation.php

4. Créer votre Frontend dans app/:
   - index.html (votre design)
   - style.css (votre CSS)
   - app.js (votre logique)

5. Tester avec Postman (postman-collection.json)
